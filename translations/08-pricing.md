# Pricing page (/pricing)

The pricing page is dense with package details, prices, and bullet lists. This file
covers all section headings, package names, "best for" descriptions, key labels,
the FAQ, and the final CTA. The repetitive "includes" bullet lists use a single
translated reference table at the end so reviewers don't have to re-read the same
terms 50 times.

Prices, currency (R), VAT note, and SLA terms are kept identical (numbers don't translate).
